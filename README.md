
# B-Market WebSite project

In this project, I built a simple E-commerce website called B-Market using the stack:

Frontend:
- HTML
- CSS
- React
- React-Redux
- Bootstrap
- React Router DOM

Backend:
- ExpressJs
- NodeJS
- MongoDB
- Redis
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/liorc955/b-market-project.git
```

Go to the project directories:

```bash
  cd frontend
```

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

## Running

On the frontend:

```bash
  npm start
```

On the backend:

```bash
  nodemon index.js
```
